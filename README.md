# 💡 Random Quote Generator

A sleek, responsive web application that generates random, inspirational quotes at the click of a button. This project fetches real-time data from a third-party API and updates the UI dynamically using intermediate JavaScript DOM manipulation.



## ✨ Features
* **Real-time API Integration:** Fetches fresh quotes instantly using the `quotable.io` API.
* **Dynamic DOM Updates:** Uses JavaScript to seamlessly inject new text and authors into the HTML structure without reloading the page.
* **Responsive Design:** Fully styled with custom CSS to ensure a clean user experience across desktop and mobile devices.
* **Asynchronous JavaScript:** Implements modern ES6+ `fetch()` (or `async/await`) to handle network requests efficiently.

## 🛠️ Built With
* **HTML5** - Semantic markup for the application structure.
* **CSS3** - Custom styling, layout design, and responsive media queries.
* **JavaScript (ES6+)** - Logic for event handling, API fetching, and DOM manipulation.

## 🔌 API Reference
This project relies on the free and open-source [Quotable API](https://github.com/lukePeavey/quotable). 
* **Endpoint used:** `https://api.quotable.io/random`
